## 📌 Overview
This project simulates a real-world e-commerce business problem focused on **customer churn and revenue leakage**.

The goal is to:
- Identify customer retention patterns
- Detect high-risk churn customers
- Estimate revenue at risk
- Evaluate ROI of retention strategies

---

## 🚀 Key Features

- 📊 Cohort Retention Analysis  
- 🎯 RFM Customer Segmentation  
- 🤖 Churn Prediction Model (Logistic Regression)  
- 💰 Revenue Leakage & ROI Analysis
