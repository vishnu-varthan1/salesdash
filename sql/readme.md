## 📌 Overview
This project simulates a real-world e-commerce analytics problem focused on **customer churn and revenue leakage**.

The goal is to identify:
- Customer retention patterns
- High-risk churn customers
- Revenue at risk
- ROI of retention strategies

---

## 🚀 Key Features

- 📊 Cohort Retention Analysis
- 🎯 RFM Customer Segmentation
- 🤖 Churn Prediction Model (Logistic Regression)
- 💰 Revenue Leakage & ROI Analysis

---

## 🛠 Tech Stack

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
